- use Postgre SQL in Codeigniter, table case sensitive must be lowecase typing
- Tips: use Backend or ServerSide to manipulate data, if use frontend to manipulate could be lagging or crash.
- Tips: use Ajax in some any trick you can do it
        - $.ajax({
            url = "your/url"
            method: "POST" or GET or ETC
            type: "JSON" or ETC
            data: {
                key: value
            }
        }).done(function(data) {
            console.log(data)
        }).fail(function(data) {
            console.log(data)
        }
        - $.post("your/url", { key:value }, function(data) {
            console.log(data)
        })
        - $.get("your/url", { key:value }, function(data) {
            console.log(data)
        })
